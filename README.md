# promini-sdcard
sdcard write test for arduino pro mini

# pin connection

| Pro Mini      | SD Module      |
|:-------------:|:---------------|
|     13        |    CLK (SCK)   |
|     12        |    MISO        |
|     11        |    MOSI        |
|     10        |    CS          |
|     3.3V(5V)  |    VCC         |
